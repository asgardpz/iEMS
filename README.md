# EV Charger Backend System

這是一個電動車充電樁的後台管理系統，提供完整的帳號與設備管理功能，支援角色與權限控管。

## 功能模組

- **Real-Time Monitoring**
  - 即時監控充電樁狀態與使用情況

- **Dashboard**
  - 系統總覽，顯示主要統計數據與指標

- **Transactions & Billing**
  - 交易紀錄查詢
  - 帳單管理與費率設定

- **Maintenance Management**
  - 維護工單建立與追蹤
  - 設備維修紀錄管理

- **Reports & Analytics**
  - 報表產生 (PDF/CSV 匯出)
  - 使用數據分析

- **Device Management**
  - 充電樁設備清單
  - 設備狀態與設定管理

- **Work Order Management**
  - 工單建立、分派與進度追蹤

- **Member Management**
  - 使用者帳號管理
  - 員工/客戶資訊維護

- **Permission Settings**
  - 角色與權限控管
  - 功能存取限制設定

- **Account Management**
  - 帳號建立、編輯、刪除
  - 密碼修改與狀態管理
  - 搜尋與篩選帳號
  - 登入紀錄查詢

## 技術架構

- **後端框架**：Laravel (PHP)
- **前端**：Blade + JavaScript + CSS
- **資料庫**：MySQL
- **版本控制**：GitHub
