# EV Charger Backend System

這是一個電動車充電樁的後台管理系統，提供完整的帳號、設備、交易與維護管理功能，並支援角色與權限控管。

---

使用者登入介面，支援帳號密碼驗證

## 技術架構

- **後端框架**：Laravel (PHP)
- **前端**：Blade + JavaScript + CSS
- **資料庫**：MySQL
- **版本控制**：GitHub
